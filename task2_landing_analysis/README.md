## 1. Анализ текущей страницы компании Визит

### Что есть на странице:
- [x] Краткое описание ипотечной программы
- [x] Форма заявки (имя, телефон)
- [x] Контактный телефон
- [x] Логотипы банков-партнёров

### Чего не хватает:
- [ ] Ипотечный калькулятор с расчётом ежемесячного платежа
- [ ] Список аккредитованных новостроек
- [ ] Блок "Часто задаваемые вопросы" (FAQ)
- [ ] Отзывы клиентов, получивших ипотеку
- [ ] Пошаговая инструкция: "Как получить ипотеку за 5 шагов"
- [ ] Сравнение условий разных банков
- [ ] Калькулятор налогового вычета
- [ ] Онлайн-чат с консультантом
- [ ] Документы для скачивания (памятка заёмщика)

### Оценка:
Страница содержит базовую информацию, но не даёт пользователю достаточно данных для принятия решения. Нет интерактивных элементов, которые удерживают внимание и повышают доверие.

**Что понравилось:**

**Что можно позаимствовать:**

- Подробное описание каждого шага оформления
- Видео-инструкции
- Реальные отзывы с оценками
- Калькулятор с возможностью сравнить программы

**Что можно позаимствовать:**
- Блок "Как оформить ипотеку" с визуальными шагами

**Что понравилось:**

- Фильтр по районам и цене
- Выделенные спецпредложения

## 3. Рекомендации по улучшению страницы

### 3.1. Добавить ипотечный калькулятор
Стоимость квартиры: ______ ₽
Первоначальный взнос: ______ ₽ (__%)
Срок кредита: ______ лет
➡️ Ежемесячный платёж: ______ ₽

Калькулятор должен пересчитывать сумму в реальном времени.

### 3.2. Добавить блок "Аккредитованные новостройки"
Таблица с фильтром:
| Жилой комплекс | Застройщик | Срок сдачи | Стоимость | Спецпредложение |
|----------------|------------|------------|-----------|-----------------|
| ...            | ...        | ...        | ...       | ...             |

### 3.3. Добавить блок "Сравнение банковских программ"
| Банк | Ставка | Первоначальный взнос | Сумма | Особенности |
|------|--------|----------------------|-------|-------------|
| Сбер | от 5.3% | от 20% | до 12 млн | Семейная ипотека |
| ВТБ | от 5.5% | от 15% | до 12 млн | На вторичку тоже |
| ... | ... | ... | ... | ... |

### 3.4. Добавить FAQ с микроразметкой
Вопросы, которые волнуют клиентов:
- Какие документы нужны для ипотеки?
- Могу ли я получить ипотеку с плохой кредитной историей?
- Что такое аккредитация новостройки?
- Как досрочно погасить ипотеку?
- Какие есть госпрограммы поддержки?

Микроразметка FAQ (Schema.org) поможет выводить вопросы прямо в поиске.

### 3.5. Добавить блок "Реальные отзывы"
3-4 отзыва с фото клиентов, их историей и результатом (какую квартиру купили).

### 3.6. Добавить пошаговую инструкцию
1. **Выбор квартиры** — подбор по параметрам
2. **Подача заявки** — онлайн-форма
3. **Одобрение** — от 15 минут
4. **Сделка** — встреча в офисе или онлайн
5. **Заселение** — получение ключей

### 3.7. Улучшить форму заявки
Добавить поля:
- Желаемая сумма кредита
- Первоначальный взнос
- Удобное время звонка
- Чек-бокс "Согласие на обработку данных"

### 3.8. Добавить элементы доверия
- Сертификаты и награды компании
- Количество выданных ипотек (статистика)
- Партнёры (банки, застройщики)
- Сертификаты сотрудников

### 3.9. Оптимизация под мобильные устройства
Проверить, чтобы на телефоне:
- Калькулятор работал корректно
- Кнопки были крупными
- Форма легко заполнялась

## 4. Ожидаемый эффект от изменений

| Метрика | Ожидаемый рост |
|---------|----------------|
| Конверсия в заявку | +30-50% |
| Время на сайте | +1.5 минуты |
| Глубина просмотра | +2 страницы |
| Отказы | -15% |

## 5. Вывод

Страница компании Визит имеет хорошую базу, но требует доработки для соответствия лучшим практикам рынка. Внедрение предложенных элементов повысит доверие посетителей, даст им больше информации для принятия решения и, как следствие, увеличит количество обращений.
