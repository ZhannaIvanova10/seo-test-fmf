# Задание №3. Сбор семантики

**Страница:** https://www.promstroysever.ru/product/stroitelnaya-tekhnika/krany/bashennye-krany/  
**Регион продвижения:** Москва  
**Тип страницы:** Категория товара (башенные краны)

## 1. Процесс сбора семантики

### Шаг 1. Определение масок запросов

Для страницы с башенными кранами я выделил основные маски (паттерны) запросов:

1. **Общие запросы**: башенный кран, башенные краны
2. **Коммерческие запросы**: купить башенный кран, продажа башенных кранов
3. **Транзакционные с ценой**: цена башенного крана, стоимость башенного крана
4. **Запросы с характеристиками**: башенный кран грузоподъемностью, высота подъема
5. **Запросы с брендами**: кран Liebherr, кран Potain (если есть в ассортименте)
6. **Запросы с регионом**: башенный кран Москва, купить кран в Москве
7. **Услуги**: аренда башенного крана, лизинг башенного крана
8. **Специфические типы**: кран с башенным оборудованием, приставной башенный кран

### Шаг 2. Сбор запросов в инструментах

Использовал следующие инструменты:
- **Яндекс.Вордстат** (wordstat.yandex.ru) — основной сбор
- **Keys.so** — для расширения семантики
- **База Пастухова** — для проверки частотности
- **Подсказки Яндекса и Google** — для сбора длинного хвоста
- **Анализ конкурентов** — смотрю, по каким запросам ранжируются топ-10 сайтов

### Шаг 3. Чистка и фильтрация

Удалил:
- Нецелевые запросы (про игрушки, детские краны)
- Запросы с частотой 0
- Явно нерелевантные (например, "кран водопроводный")
- Дублирующиеся по смыслу

### Шаг 4. Группировка

Разбил запросы на группы по интенту пользователя:
- Информационные (что такое, как выбрать)
- Коммерческие (купить, цена)
- Транзакционные (аренда, заказать)

## 2. Инструменты для сбора семантики

| Инструмент | Назначение | Бесплатный доступ |
|------------|------------|-------------------|
| Яндекс.Вордстат | Сбор частотности, подбор ключей | Да, с ограничениями |
| Keys.so | Кластеризация, подбор по конкурентам | Есть демо-доступ |
| База Пастухова | Парсинг Вордстата | Да, 500 запросов/день |
| Пиксель Тулс | Сбор семантики, анализ конкурентов | Есть бесплатный тариф |
| Google Keyword Planner | Подбор ключей (для контекста) | Да, через Google Ads |

## 3. Критерии включения запросов в семантику

✅ **Включаем запросы, если:**
- Частотность в Москве ≥ 10 показов в месяц
- Запрос релевантен товару (башенные краны)
- Запрос имеет коммерческий или информационно-коммерческий интент
- Запрос соответствует типу страницы (категория товара)
- В запросе есть явное указание на покупку/цену/характеристики

## 4. Критерии НЕвключения запросов

❌ **Исключаем запросы, если:**
- Запрос содержит слова "игрушка", "детский", "лего", "модель" (не наш товар)
- Запрос про водопроводные или сантехнические краны
- Частотность в регионе = 0
- Запрос на иностранном языке без локализации
- Запрос с явными ошибками, не влияющими на поиск
- Информационные запросы, не ведущие к покупке ("история башенных кранов")

## 6. Рекомендации по использованию

1. **Для страницы категории** использовать коммерческие запросы в Title, H1, описании
2. **Для карточек товаров** использовать уточняющие запросы (с грузоподъемностью, брендами)
3. **Для статей блога** использовать информационные запросы
4. **В мета-теги** включить топ-3 запроса с регионом
5. **В текст** включить все ключи равномерно, избегая спама
