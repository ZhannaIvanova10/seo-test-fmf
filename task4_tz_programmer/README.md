### 3.4. Отображение товаров

На странице должны выводиться только те товары категории "Коленчатые подъемники", у которых характеристика **"Грузоподъемность"** > 300 кг.

**Условия фильтрации:**

- Значение характеристики "Грузоподъемность" > 300 кг
- Сортировка: по умолчанию (по популярности или цене)

### 3.5. Внешний вид страницы

Страница должна быть выполнена в том же стиле, что и существующие подборки (например, `/products/kolenchatye-podemniki/`).

**Элементы на странице:**
1. **Шапка:** хлебные крошки, заголовок H1
2. **Фильтры (опционально):** если есть техническая возможность, добавить фильтры по:
   - Высоте подъема
   - Цене
   - Бренду
3. **Сетка товаров:**
   - Фото товара
   - Название (ссылка на карточку)
   - Цена
   - Краткие характеристики (грузоподъемность, высота)
   - Кнопка "Заказать" или "В корзину"
4. **Пагинация** (если товаров много)
5. **SEO-текст** (будет добавлен позже контент-менеджером)
